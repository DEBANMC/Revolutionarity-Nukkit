Revolutionarity
-

Bedrock: 1.18.2
PowerNukkit: 1.0.13

Revolutionarity is a bad and basic AntiCheat for PowerNukkit, providing checks against:
- AirJump
- AntiImmobile
- BlockReach
- FlightA (Value check)
- FlightB (Spider will get flagged by this)
- FlightC (Illegal flying player, PlayerToggleFlightEvent)
- Glide
- KillAura
- NoSwing
- Reach
- SelfHit
- Speed (Broken by Nukkit)
